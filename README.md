# PokeDex App
Aplikasi untuk menampilkan daftar pokemon dan detail pokemon yang didapat dari API https://pokeapi.co/docs/v2

# Feature
- GetX
- Dio
- MVC
- Lottie Animation
- Infinite Scroll Pagination
